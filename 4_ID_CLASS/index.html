ID E CLASS

o ID funciona como um cpf. assim como so posso ter um numero de cpf cada elemrnto pode ter um valor de id. o cpf e unico ou seja, ninguem pode ter cpf igual ao meu. isso significa que o td tambem é unico, entao é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que voce pode ter um valor, no class voce pode ter varios valores. tambem diferente do ID, em que nao se pode repetir o valor, no class voce pode utilizar o mesmo valor para varios elementos.
